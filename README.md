# CoffeeApp

* Allow Ordering Coffee with Different Toppings :coffee: 
* Allows sending an order-summary email :e-mail:
## Pre-requisites

    Android SDK v24
    Android Build Tools v24.0.2
    Android Support Repository v24.2.0
